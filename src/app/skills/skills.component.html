Project description :

TalkingThreads – my pride and joy – full-featured WebRTC-capable application based on OpenVidu platform, which can be used for creating video-call sessions.

This application is designed for those who want be in touch with their friends, relatives and colleagues but don’t like to install additional software.

TalkingThreads provides video, voice and text communications. And all one needs to use it is Internet regardless of browser! And of course, TalkingThreads is completely free to use.
